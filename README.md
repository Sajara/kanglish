# Kanglish

A lightweight, offline-capable web app that translates English or Tamil (transliterated) into conversational Bengaluru Kanglish.

**Live:** https://sajara.github.io/kanglish/

## Features
- Type in English or transliterated Tamil
- Auto-detects input language
- Outputs natural Kanglish in English script
- Works fully offline — no APIs, no tracking
- Mobile-first design
